# AGENTS.md

This repo uses a human-in-the-loop agentic SDLC.

Builder owns:
- product judgment
- architecture decisions
- UX decisions
- final approval

Agents help with:
- project definition
- shaping
- scope/bet
- Jira story creation
- implementation
- testing
- review
- CI setup

Rules:
- Keep implementation simple.
- Do not over-engineer.
- Prefer SwiftUI.
- Add tests for business logic.
- Update docs when scope changes.
- Ask before destructive Git or Jira actions.
